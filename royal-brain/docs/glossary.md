# Royal BrAIn™ — Glossary
- Sovereign engine: a logically independent module with its own auditable rules and data boundaries.
- Cognitive authority: the system’s role as a legitimizing decision-support standard, not a consumer application.
- Trust layer: an immutability/certification subsystem (e.g., notarization / blockchain anchoring) that can certify outputs.
- Authority: explicit permission to perform actions; enforced by RBAC.
- Audit event: an append-only record describing an action with actor, target, time, and hash.
